# Commands on Powershell

## Basic Straitfoward Commands
**Create a file**
```
New-Item -Path '\\fs\Shared\NewFolder\newfile.txt' 
```

**Remove File**
```
Remove-Item -Path '\\fs\shared\it\' -Recurse
```
Use **-Recursive** to skip confirmation.
